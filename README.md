**[Macro Regime & Asset Performance Analytics](https://github.com/ml20g2l/Macro-Financial-Datasets)**

Built an end-to-end Python and PostgreSQL pipeline that classified 1,111 trading days into four transparent, one-day-lagged macro regimes without look-ahead bias. Evaluated SPY, IEF, and GLD across return, volatility, drawdown, Sharpe ratio, and cross-asset correlation, reconciling all 24 metric and correlation outputs between Python and PostgreSQL with zero discrepancies. A self-contained Tableau dashboard and robustness testing across six threshold scenarios and 5,000 moving-block bootstrap resamples confirmed GLD delivered the highest Tightening-regime return in all six scenarios while highlighting uncertainty in the limited Stress sample.

`Python` `PostgreSQL` `Tableau` `Statistical Analysis`
