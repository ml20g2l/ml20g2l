### 🚀 Featured Projects
**🤖 [AI-Augmented Customer Support Operations Analytics](https://github.com/ml20g2l/AI-Augmented-Customer-Support-Operations-Analytics)**
Validated Gemini-assisted ticket classification and designed an empirical routing policy, supported by a three-page Power BI dashboard for operational impact and root-cause analysis.

`Python` `Gemini API` `Power BI` `scikit-learn`

**📊 [Customer Behaviour & Conversion Analytics Dashboard](https://github.com/ml20g2l/E-Commerce-Product-Analyics-Conversion-Funnel-Retention-Revenue-Drivers)**
E-commerce analytics on 4.3M+ customer sessions and 157K+ purchase sessions using PostgreSQL and Power BI. Built a six-page interactive dashboard covering conversion funnels, retention, and product performance, and surfaced 310K high-intent lost sessions to prioritise 5 A/B test proposals.

`PostgreSQL` `Power BI` `Data Analysis`

**📈 [Macro Regimes and Asset Performance Analysis](https://github.com/ml20g2l/macro-financial-datasets)**
Integrated 5+ macroeconomic and financial datasets spanning 15+ years using PostgreSQL and Tableau. Built regime classification logic to analyse how equities, bonds, and gold performed through events like the GFC and COVID-19.

`SQL` `Tableau` `Data Analysis`

**🏠 [London Rent Affordability Analysis Dashboard]((https://github.com/ml20g2l/London-Rent-Affordability-Analysis))**
Analysed 4,000+ city-level income and rental records to build affordability metrics (rent burden, residual income) with an interactive Tableau dashboard, revealing locations offering £3,300+ more disposable income per month.

`SQL` `Excel` `Tableau`

**💼 [UK Job Market Analysis Dashboard](https://github.com/ml20g2l/uk-job-market-analysis-dashboard)**
Cleaned and analysed 2,285 UK job postings with PostgreSQL and Excel, identifying 34 duplicate groups and building a PivotTable/Power Query dashboard to explore hiring trends and salary patterns.

`PostgreSQL` `Excel` `Power Query`
